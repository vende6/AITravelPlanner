Feller AG User Management Portal

Centralized identity management and authentication solution for Switzerland's leading electronics manufacturer

Project Overview

Working as part of a cross-functional team, I contributed to the development of a centralized user management portal for Feller AG, the leader in the Swiss electrical installation market and a Schneider Electric company. The solution delivers identity management for end users while also serving as an integration platform between Feller's applications and services.

The platform provides Single Sign-On (SSO), centralized user management, and delegated administration capabilities, which form the backbone of Feller's digital ecosystem. This solution enables secure and seamless access to various products and services while maintaining strict data protection standards.

The Challenge

Feller AG faced several challenges with their existing user authentication and management systems:

  • Multiple disconnected applications with separate user databases
  • Lack of centralized user management capabilities
  • No ability for company administrators to manage their users
  • Inconsistent user experience across different platforms
  • Limited real-time analytics on user activity and behavior
"Feller sought an Identity partner to help define, design and develop a new platform for SSO, delegated company administration and real-time analytics."

Our task was to create a robust identity management solution that would serve as the foundation for Feller's digital transformation efforts, enabling seamless integration between their various applications while providing enterprise-grade security.

The Solution

We developed a comprehensive identity and access management solution with Auth0 at its core. The system provides:

  • Single Sign-On (SSO) across all Feller applications and services
  • Centralized user management with role-based access control
  • Delegated administration for company administrators
  • Real-time analytics and reporting on user activity
  • Integration with existing systems through APIs

The solution architecture placed Auth0 at the center, providing enterprise-grade Identity-as-a-Service (IDaaS) for all applications, products, and platforms within the Feller ecosystem. This enabled a microservice architecture for user management within the business and facilitated data sharing between applications.

My Role & Contributions

As a Software Engineer on this project, my responsibilities included:

  • Development of the centralized user management portal web application
  • Maintenance of the customer identity and access management (CIAM) solution provided by Auth0
  • Working with Auth0 management platform across multiple tenants
  • Implementing Rules and Hooks as part of the authentication pipeline
  • Integration with Amazon Web Services (AWS)

Technology Stack

React React Native Node.js Auth0 AWS Serverless Architecture S3 CloudFormation CloudFront RDS IAM SNS DynamoDB Xamarin

Results & Impact

The implementation of the User Management Portal delivered significant benefits to Feller AG:

  • Unified user experience across all digital touchpoints
  • Reduced administrative overhead through self-service and delegated administration
  • Improved security through centralized authentication and authorization
  • Enhanced visibility into user activity and behavior
  • Simplified integration of new applications into the ecosystem
  • Enabled data sharing between applications to create a more connected experience

The solution now serves as a critical foundation for Feller's digital ecosystem, enabling them to deliver seamless, secure, and personalized experiences to their customers across all digital touchpoints.
